Intro

Estudante de Análise e Desenvolvimento de Sistemas no IFPI, atualmente no bloco 5. Originário de São Paulo, estou em Parnaíba há 2 anos. Anteriormente, atuei na área elétrica, onde realizei dois cursos no SENAI. Hoje, estou focado na programação, especialmente em React, React Native, Node.js e Python, criando CRUDs, APIs e interfaces com base em UX e UI. Minhas próximas metas incluem dominar as linguagens em que estudo para avançar para o desenvolvimento móvel. Recentemente, participei de um projeto ajudei a desenvolver um aplicativo móvel para agendamento de quadras de beach tennis, atuando principalmente no frontend.

Experiência Profissional

Beach Time

Atuei principalmente no desenvolvimento front-end utilizando React Native, criando interfaces interativas e responsivas para o aplicativo Beach Time, um projeto voltado ao gerenciamento de agendamentos em quadras de beach tennis. Essa experiência foi importante para aprofundar minhas habilidades em desenvolvimento mobile.

LUCIA

Trabalhei tanto no front-end quanto no back-end, desenvolvendo funcionalidades completas e participando ativamente do ciclo de vida dos produtos.

SN_REST_API

Trabalhei com Docker, Python, Django Ninja e autenticação JWT no projeto SN_REST_API. Esse projeto envolveu a criação de uma API REST robusta, com endpoints seguros e estrutura de autenticação avançada, utilizando Django Ninja para simplificar a implementação e garantir alta performance na comunicação entre serviços.
